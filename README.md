# Bank System Project in Python

This is a simple **Bank Management System** built in Python using **Streamlit**.  
Users can add, view, and manage bank accounts through an interactive web interface.

---

## Features

- Add new bank accounts
- View account details
- Manage deposits and withdrawals
- User-friendly web interface with Streamlit

- Streamlit
